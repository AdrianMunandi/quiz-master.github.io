# Quiz Website

This Quiz Website is a web application that allows users to participate in quizzes and test their knowledge on Front-end Languages.

# Usage
To use the Quiz Website, follow these steps:
Clone the repository or download the code files to your local machine.
Set up a local server environment to run the application.
Copy the code files to the appropriate directory within your web server environment.
Start the server and ensure it is running and accessible.
Open a web browser and enter the URL for the Quiz Website
The Quiz Website should load in the web browser, and you can now start participating in the quizzes by selecting the answer options and clicking the submit button.
After submitting your answers, the website will provide immediate feedback on your responses and display your score or results.

# Technologies Used
The application is built using the following technologies:

HTML
CSS
JavaScript
Fetch API for making HTTP requests to the server
Database json
